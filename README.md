# Network Config Generator
This tool automates the creation of network device configurations.
It separates **data** (YAML) from **logic** (Jinja2 templates) to ensure standardization.

## Tech Stack
- Python 3
- Jinja2 (Templating)
- YAML (Data Serialization)

## Usage
Run `python3 generate.py` to build configs for all devices defined in `routers.yaml`.
